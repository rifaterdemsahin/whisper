Quick Start ‚Äî Local Turkish Transcription

# Install Whisper
pip install openai-whisper

# Transcribe a video in Turkish (medium model recommended)
whisper "video.mp4" --model medium --language Turkish

# For large files: extract audio first
ffmpeg -i video.mp4 -q:a 0 -map a audio.wav
whisper audio.wav --model large --language Turkish --output_format srt
